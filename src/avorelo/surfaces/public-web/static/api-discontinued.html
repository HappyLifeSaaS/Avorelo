This hosted endpoint has been discontinued

The hosted Avorelo API and the service behind it have been discontinued. No hosted endpoint replaces this one, and there is no account, billing, or cloud backend to connect to.

Avorelo is now local-first: it runs entirely on your own machine, and the state and evidence it produces stay there.

Avorelo is now Open Source under the Apache License 2.0. Personal, internal, organizational, and commercial use are all permitted under Apache-2.0. Install with npm install -g avorelo. See the license page.

Support bundles are created locally with avorelo support bundle — nothing is uploaded, and you decide whether to share the result.
Report bugs at the public GitHub issues; for vulnerabilities follow the security policy. Please do not post sensitive information publicly.
