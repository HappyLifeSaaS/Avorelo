Avorelo has current public-claim registers, product-claim substantiation docs, and review helpers that keep copy honest. In 0.3.0 that support is real, but it is still primarily a governed review layer rather than a dedicated runtime execution pack.

Tests

Public copy and public-surface truth are checked in the site and journey suites rather than a dedicated PMM runtime executor.

What exists today

Current copy is bounded by claim registers, legal-review notes, and static-surface tests that block unsupported claims about launch, billing, security, savings, or autonomy.

Claim boundary

This support lane reviews and constrains messaging. It does not auto-approve every public claim, and it does not replace a product marketer, technical writer, or legal reviewer.

What it does not do

It does not guarantee conversion outcomes, and it does not claim a full autonomous content review engine is already shipping in the current runtime.
